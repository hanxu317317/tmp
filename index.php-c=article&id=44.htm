适合新手的网页制作工具有那些

2018-04-27

制作网站已经不再是我们所知道的DreamWeave、Fireworks、HomeSite等复杂的制件软件了，我们完全可以借助一些专业的网站管理系统，普通用户也可以轻松的建立出相对专业的网站，比如建企业网站，可以去下载PageAdmin企业建设系统，建购物网站可以去下载ecshop系统、建论坛网站可以去下载discuz系统，借助这些系统，我们就不需要任何编程基础，也不需要专业的技术知识，一个从来不懂做网站的人稍微花点时间熟悉，一天也能搭建好一个网站。

当然网站制作中也会用到一些专业网站制作软件，如图片软件，网站虽然可以下载建站系统来完成，但是网站上的一些图片，比如网站的logo，横幅等等这些就需要我们通过图片软件来完成，常用的图片软件有Firework和photoshop，掌握一个就足够了。

1、Firework是Adobe推出的一款网页作图软件，软件可以加速 Web 设计与开发， 是一款创建与优化 Web 图像和快速构建网站与 Web 界面原型的理想轻量级工具，可以用于快速制作或处理网页上的图片。

2、Adobe Photoshop，简称“PS”，是由Adobe Systems开发和发行的图像处理软件，Photoshop主要处理以像素所构成的数字图像，使用其众多的编修与绘图工具，可以有效地进行图片编辑工作，ps有很多功能，在图像、图形、文字、视频、出版等各方面都有涉及。相比Firework，Photoshop就专业很多，但是也复杂很多，专业的平面图片设计人员必须掌握的一款工具。

3、Pageadmint是一个独特有个性的系统，扩展性强，它没有采用主流的html模板的方式来设计，而是采用后台布局和框架组合式的设计，很容易上手。我个人是很喜欢这系统，从2.1到3.0我基本都用过，一路发展下来，功能不断给用户惊喜，自定义表单和自定义模型这块设计得很好，我曾用这个系统给各类做过网站管理系统。如果对asp.net熟悉，你可以直接下载进行建站，也可以用这个系统扩展出很多功能。 适用企业、学校、政府或个人网站 运行环境：Windows+IIS+Framework2.0(或以上）

4、免费开源的独立网店系统，如果你懂技术可以很轻松对ecshop进行二次开发，ecshop、shopex和ecmall都是商派旗下的开店软件，后续开发完全不用考虑。 适用购物网站，商城类网站 运行环境：windows+iis或linux+apache都可以

5、最受欢迎的论坛系统，功能强大，做论坛的首选，模板众多，论坛十分活跃。被腾讯收购，Discuz X 更是引入QQ登陆，SOSO的纵横搜索，非常实用的功能。 无论是做网店、门户、分类信息、威客、团购DZ都是很好的搭档。 适用论坛，社区型网站 运行环境：windows+iis或linux+apache都可以
